📍 新羅舒泰優勢： 步行 5 分可達「漢拏醫院站」公車樞紐。
📸 攝影核心： 尋找薄荷漸層海與火山岩極簡線條。
08:00 健身與甦醒

飯店 1F 健身房運動 1 小時，隨後清爽出發前往「東福里」。

10:00 人氣早午餐排程
Must Go
倫敦貝果博物館 (濟州店)
現象級名店，建議選「外帶」至海邊食用，節省 2 小時排隊時間。
📸 構圖：店外牆面的英倫標誌與濟州大海交界。
Naver Map
11:20 早晨咖啡 (平行備選) 左滑選擇 👉
海景首選
Cafe Mou Moon (月汀里)
巨大的二樓落地窗，將月汀里的薄荷藍漸層海完整框住。
📸 桌布位：退後 3 米，將大窗作為桌布邊框。
Naver Map
Cafe Layered
英式甜點名店，若貝果人潮過多，這裡是不必移動的最佳備案。
Naver Map
12:45 獨食午餐方案
明珍鮑魚 (명진전복)
單人鮑魚石鍋飯名店。飯後加入熱水製作營養豐富的鍋巴湯。
Naver Map
벵디 (Bengdi)
整隻巨大章魚蓋飯，設有面海單人吧台，視覺衝擊力極強。
Naver Map
14:15 特色咖啡與世界大景
Dorrell Coffee (도렐)
工業風特色空間，必點 Nutty Cloud 堅果雲朵拿鐵 (限時16:00前)。
Naver Map
城山日出峰 (左側步道)
不登頂，走海女之家旁的海岸線，拍出黑色火山岩的壯闊感。
📸 廣角拍攝：黑色玄武岩與白色浪花的對比。
Naver Map
17:45 咸德夕陽與獨食晚餐
咸德馬當 (함덕마당)
極其罕見的單人份辣燉白帶魚套餐。就在咸德海灘旁。
📸 攝影：先爬上犀牛峰步道，捕捉日落前海水的三種藍。
Naver Map 導航
08:00 晨間健身 / Check-out

最後一小時運動，帶著行李前往「漢拏醫院站」搭 102 急行公車。

10:00 廢墟美學咖啡
Industrial
Anthracite Coffee (翰林)
澱粉工廠改建，蕨類與鏽蝕機具共生。極致孤獨冷淡風。
📸 桌布：拍攝光影從舊廠房頂棚灑下的線條。
Naver Map
明月國民學校
廢校改建。復古走廊與木質教室，懷舊氣息十足。
Naver Map
11:15 桌布級大景：金陵海灘
Desktop Shot
飛揚島 (Biyangdo) 視角
站在白沙灘上拍攝海中的孤島。這是西線最具標誌性的海景。
📸 構圖：利用長焦段將飛揚島與近處浪花壓縮在一個畫面。
Naver Map
12:30 獨食午餐方案
Nolaba 海鮮拉麵 (노라바)
螃蟹拉麵配面海吧台座位。一人享用美食與無敵海景。
Naver Map
西村祭 (黑豬肉炸豬排)
厚切多汁，完全歡迎單人客，份量十足。
Naver Map
14:30 海邊的靜謐藝廊
Slowboat (涯月)
藝廊式咖啡空間。大片玻璃窗將海景裁切成一幅永恆的畫。
📸 必拍：二樓看向窗外的「畫框海景」。
Naver Map
Lazy Pump
舊幫浦廠改建，方形巨型觀景窗，極度適合出片。
Naver Map
17:00 旅途終章：奢華汗蒸
ULTIMATE RELAXATION

濟州君悅酒店 汗蒸幕 (6F)

在 22:00 起飛前，享受 3 小時最高規洗浴。清爽淋浴、泡熱海水、並在裡面享用最後一頓韓式晚餐。
